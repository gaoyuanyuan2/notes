# 基础

##  常见的字符集:

* ASCII

美国人编码,使用7位来对美国常用的字符进行编码-包含128个字符

* ISO-8859-1

欧洲的编码,使用8位一包含256个字符

* GB2312
* GBK

国标码,中国的编码
      
* Unicode

万国码,包含世界上所有的语言和符号,编写程序时般都会使用Unicode编码Uni 、code编码有多种实现, UTF-8 UTF-16 UTF-32 最常用的就是UTF-8

## 编译型语言和解释型语言

* 编译型语言  C语言

编译型语言,会在代码执行前将代码编译为机器码,然后将机器码交由计算机执行


*  解释型语言 Python JS Java

边执行一边编译
